A tiny plugin for pliugin 'Gutenberg'.

== Description ==
This plugin does 2 things onyl when Gutenberg is active.
1. Adding 'Classic Editor' option under 'Add New' button in post.php screen.
2. Adding 'Add new (classic)' link to admin menu.

== Installation ==

1. Upload the 'post-new-buttons-menu-using-classic-editor-for-gutenberg' folder to the '/wp-content/plugins/'' directory.
2. Activate the plugin through the Plugins menu in WordPress.
